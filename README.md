# KMS

Proyecto reorganizado con una línea visual unificada y estilos compartidos.

## Estructura principal

- `inicio.html` → panel principal del proyecto
- `shared/` → estilos y scripts compartidos
  - `theme.css` → variables, fondo, botones, inputs, shell general
  - `home.css` → paneles principales y tarjetas de navegación
  - `forms.css` → calculadoras simples
  - `hilo.css` → calculadora de consumo de hilo
  - `accessories.css` → módulo de accesorios
  - `home.js` → partículas decorativas y animación de tarjetas
- `calculadora/` → módulos operativos
- `Accesorios/` → generador de Excel para compras y tags

## Cambios aplicados

- Visual unificado basado en el estilo futurista del módulo de hilo
- Diseño responsive para escritorio, tablet y móvil
- Navegación más clara entre módulos
- Corrección de rutas inconsistentes
- Mejor jerarquía visual y espaciado
